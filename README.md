## ☁️ Licenciamiento de Azure: Modelos, Servicios y Plan de Aprendizaje

 Este repositorio es una guía técnica especializada sobre el ecosistema de licenciamiento de Microsoft Azure. Proporciona un marco estructurado para comprender modelos de costes, optimización de servicios y estrategias de ahorro, condensado en un itinerario de aprendizaje de 60 minutos.
